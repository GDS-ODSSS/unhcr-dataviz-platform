Tools

To support UNHCR staff to create high-quality brand-compliant charts and graphics, GDS created a number of tools, templates and scripts. You can find useful tools to support your data visualization using Excel, Power BI, Adobe Illustrator, R, Python, D3, and for GIS.

MS Office

MS Office offers a range of tools for communication, presentation and data visualization. They all include various types of charts like Column charts, Bar charts and Pie charts, and much more. In this section, you will find customized themes and chart templates that will help to avoid the default style and ease the creation of UNHCR branded charts.

Power BI

Power BI is a collection of software services, apps, and connectors that work together to turn your unrelated sources of data into coherent, visually immersive, and interactive insights. Power BI Desktop is a desktop application you can install on your computer, that works cohesively with the Power BI service by providing advanced data exploration, shaping, modeling, and report creation with highly interactive visualizations.
