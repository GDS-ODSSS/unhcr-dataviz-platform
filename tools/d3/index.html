D3 Charts

D3.js (Data Driven Documents) is an open source JavaScript library that allows you to create custom interactive data visualizations in a web browser using SVG, HTML and CSS. With the library you can create standard visualizations like line charts, bar charts, and pie charts. Because of the powerful visualization components and the flexibility of library, you can also create custom and complex visualizations.
